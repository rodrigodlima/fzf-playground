5Jenkins CLI Setup

üìã Complete setup guide for Jenkins CLI integration
Download Jenkins CLI
wget http://YOUR_JENKINS_URL/jnlpJars/jenkins-cli.jar
Alternative: curl download
curl -O http://YOUR_JENKINS_URL/jnlpJars/jenkins-cli.jar
Set Jenkins URL (temp)
export JENKINS_URL=http://YOUR_JENKINS_URL
Add to .bashrc/.zshrc (permanent)
echo 'export JENKINS_URL=http://YOUR_JENKINS_URL' >> ~/.bashrc
üîê Authentication Setup - Generate API Token
Step 1: Go to Jenkins User Config
Open: YOUR_JENKINS_URL/user/YOUR_USERNAME/configure
Step 2: Generate API Token
Click "Add new Token" ‚Üí Copy the generated token
Set credentials (temp)
export JENKINS_USER_ID=your_username:your_api_token
Add credentials to .bashrc/.zshrc
echo 'export JENKINS_USER_ID=username:token' >> ~/.bashrc
Test connection
java -jar jenkins-cli.jar -auth $JENKINS_USER_ID who-am-i
List available commands
java -jar jenkins-cli.jar -auth $JENKINS_USER_ID help
üí° Pro Tip: Create an alias for easier usage
Create jenkins-cli alias
echo 'alias jcli="java -jar ~/jenkins-cli.jar -auth $JENKINS_USER_ID"' >> ~/.bashrc
Usage with alias
jcli list-jobs | fzf --preview 'jcli console {}'

6Jenkins + FZF - Real Use Cases

Select Jenkins job and view log
java -jar jenkins-cli.jar -auth user:token list-jobs | fzf --preview 'java -jar jenkins-cli.jar -auth user:token console {}'
Select job and specific build number
JOB=$(java -jar jenkins-cli.jar -auth user:token list-jobs | fzf) && java -jar jenkins-cli.jar -auth user:token builds "$JOB" | fzf --preview "java -jar jenkins-cli.jar -auth user:token console $JOB {1}"
Search failed builds with logs
java -jar jenkins-cli.jar -auth user:token list-jobs | fzf --preview 'java -jar jenkins-cli.jar -auth user:token console {} | tail -100'
Interactive job trigger
java -jar jenkins-cli.jar -auth user:token list-jobs | fzf | xargs -I {} java -jar jenkins-cli.jar -auth user:token build {}
Search in YAML/JSON
find . -type f \( -name "*.yaml" -o -name "*.json" \) | fzf --preview 'cat {}'
Active processes
ps aux | fzf
