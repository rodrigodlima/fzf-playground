3File Preview

Basic preview
cd ~/git; fzf --preview 'cat {}'
Preview with bat
fzf --preview 'bat --color=always {}'
Preview with bat and edit with vim
vim $(fzf --preview 'bat --color=always {}')
Preview and edit with larger window
vim $(fzf --preview 'bat --color=always {}' --preview-window=right:60% --height=100%)

5Git Integration

Select branch
git branch | fzf
Interactive checkout
git checkout $(git branch | fzf | tr -d ' ')
View modified files
git status --short | fzf
Search in history
git log --oneline | fzf --preview 'git show {1}'

7Use Cases - DevOps

Search .tf files
find . -name "*.tf" | fzf --preview 'bat --color=always {}'
Jenkins logs
ls -t /var/log/jenkins/*.log | fzf --preview 'tail -50 {}'
Search in YAML/JSON
find . -type f \( -name "*.yaml" -o -name "*.json" \) | fzf --preview 'cat {}'
Active processes
ps aux | fzf

8Advanced Features

Multi-select

Open in editor
