4Git Integration

Interactive checkout
git checkout $(git branch | fzf | tr -d ' ')
Search in history
git log --oneline | fzf --preview 'git show {1}'

5Use Cases - DevOps

Jenkins logs
ls -t /var/log/jenkins/*.log | fzf --preview 'tail -50 {}'
Search in YAML/JSON
find . -type f \( -name "*.yaml" -o -name "*.json" \) | fzf --preview 'cat {}'
Active processes
ps aux | fzf
